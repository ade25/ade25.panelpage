ade25.panelpage Installation
----------------------------

To install ade25.panelpage using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ade25.panelpage`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ade25.panelpage
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
